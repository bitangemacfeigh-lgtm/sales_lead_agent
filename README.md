\# 💼 Sales Lead AI Agent



An intelligent sales assistant powered by \*\*Mistral Large\*\* and \*\*Streamlit\*\*. This agent researches prospects, identifies psychological pain points, and drafts high-conversion outreach emails.



\## 🚀 Features

\- \*\*Deep Research\*\*: Analyzes company funding, role challenges, and industry trends.

\- \*\*Strategic Mapping\*\*: Identifies "Just-in-Time" vs "Just-in-Case" logic and other complex business triggers.

\- \*\*AI Copywriting\*\*: Generates professional, low-friction cold emails.



\## 🛠️ Setup

1\. Clone the repo: `git clone https://github.com/your-username/sales-lead-agent.git`

2\. Install dependencies: `pip install -r requirements.txt`

3\. Add your `MISTRAL\_API\_KEY` to a `.env` file.

4\. Run the app: `streamlit run app.py`
